Common Text Tag Documentation

The 'Name' field of the tables show the raw formatting of each tag (no additional CSS formatting)

Header Tags

Tag Name Description
<h1>

Header 1 Tag

 Formats text the largest, and with the most importance in SEO
<h2>

Header 2 Tag

 Formats text the next largest, smaller than <h1>
<h3>

Header 3 Tag

 Formats text the next largest, smaller than <h2>
<h4>

Header 4 Tag

 Formats text the next largest, smaller than <h3>
<h5>
Header 5 Tag
 Formats text the next largest, smaller than <h4>
<h6>
Header 6 Tag
 Formats text the smallest out of the headers


Other Text Tags

Tag Name Description
<b> and <strong> Bold & Strong

Although they look similar, the bold tag is a physical tag that stands for bold text, the strong tag is used to emphasize the importance of the text
<i> and <em> Italic & Emphasis

Both tags define italic text. the italic tag is only responsible for the appearance of text, while the emphasis tag adds semantic importance
<mark> Mark Tag

Highights the text
<small> Small Tag Small Tag

Decreases the text font size by one size smaller than the default font size (from medium to small, or from x-large to large). Notice the impact a small tag has with the first reference of 'Small Tag' in the name field vs the second reference that is the default text
<del> & <s> Delete Tag & Strikethrough Tag

Both formatted as strikethrough text, the delete tag indicates part of the text that was deleted from the document while the strikethrough tag defines text that is no longer correct or relevant
<sub> & <sup> Subscript Tag & Superscript Tag

Formats text as subscript or superscript
